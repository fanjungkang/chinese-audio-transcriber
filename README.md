# 🎤 中文音频转逐字稿工具

一个免费在线工具，可将中文音频文件转换为带时间轴的逐字稿，支持多种格式下载。

## ✨ 功能特性

- 🎵 **支持多种音频格式**: MP3, WAV, M4A, OGG
- ⏱️ **自动时间戳**: 每句话都带有精确的时间标记
- 📝 **实时预览**: 在转写过程中可预览结果
- 📊 **多格式导出**: 支持 TXT, SRT, JSON 格式下载
- 🎚️ **可调参数**: 可设置语言、识别精度等选项
- 📱 **响应式设计**: 在手机和电脑上都能良好显示
- 🔒 **隐私保护**: 所有处理在浏览器本地完成（演示版）
